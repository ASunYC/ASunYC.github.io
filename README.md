ASunYC.github.io is the public homepage for ASunYC projects, agent skills, and interactive platform experiments.

## Online Pages

- Homepage: https://asunyc.github.io/
- Skills Book: https://asunyc.github.io/skills-book/
- Skills Hot: https://asunyc.github.io/skills-hot/
- Skills Shop: https://asunyc.github.io/skills-shop/
- MapNews: https://asunyc.github.io/map-news/

## What Is Included

- **Home**: project hub and personal homepage.
- **Skills Book**: introduction and usage guide for the Skills Book agent skill marketplace.
- **Skills Hot**: popular and fresh skill rankings exported from Skills Book.
- **Skills Shop**: 3D globe marketplace for agent skills.
- **MapNews**: global news globe and publishing experiment.

## Local Development

```bash
npm install
npm run dev
```

Build the static site:

```bash
npm run build
```

Preview the build:

```bash
npm run preview
```

## Data

Skills data is generated by the `skills-book` project and exported into `docs/public/data`.

MapNews supports static fallback data and optional Supabase-backed dynamic content.
